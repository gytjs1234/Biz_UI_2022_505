추가적인 input 태그

number 태그 숫자만 입력 할수있는 태그
date 태그 날짜를 선택할 수 있는 달력 태그
month 태그 월을 선택할 수 있는 달력 태그
time태그 시각를 선택할 수 있는 달력 태그
button태그 앙무런 기능이 없는 버튼형태의 태그

명명( ): 함수 명명된 함수를 호출한다(매소드라고부름)

window.open(url)

window.open(url,창이름,창속성)

창이름:창이름,target을 지정할수있다.

창이름에 _self, _blank 를 지정하면 target이 적용되고 다른 형태의 이름을 지정하면 _blank(기본) 적용이된다.
